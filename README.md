# 🌌 TaskSphere

A modern MERN Stack Productivity Dashboard for managing tasks efficiently.

## Features

- User Authentication (JWT)
- Register & Login
- Create Tasks
- Delete Tasks
- Complete Tasks
- Search Tasks
- Task Categories
- Priority Levels
- Due Date Tracking
- MongoDB Atlas Integration
- Responsive Dashboard UI

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT
